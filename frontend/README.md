# NYRA – E-Commerce Saree Store

NYRA is a full-stack e-commerce web application designed for browsing and purchasing traditional Indian sarees online. The platform allows users to explore different saree collections, search for products, add items to the cart, and place orders through a modern and responsive interface.

## Features

- Browse saree collections such as daily wear and silk sarees
- Search and filter products
- User authentication with Firebase
- Add products to cart and manage cart items
- Secure checkout process
- Order success and order tracking pages
- Responsive design for desktop and mobile devices

## Tech Stack

**Frontend**
- Next.js (React)
- TypeScript
- CSS / Tailwind

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB

**Authentication**
- Firebase Authentication

## Installation & Setup

### Clone the repository

```bash
git clone https://github.com/your-username/nyra-ecommerce.git
```

### Install backend dependencies

```bash
cd backend
npm install
npm start
```

### Install frontend dependencies

```bash
cd frontend
npm install
npm run dev
```

The frontend will run on:

```
http://localhost:3000
```
